# @napl-lang/binary-linux-x64-gnu

Platform binary package for **Linux x64 glibc (x86_64-unknown-linux-gnu)**, an optional dependency of
[`napl-lang`](https://www.npmjs.com/package/napl-lang). npm installs it only on a
matching host via the `os`/`cpu` fields. The `napl` binary is dropped into this
package by the release workflow at publish time; do not install this package
directly.
